# Export Branding (PDF)

This document defines the canonical Aptale PDF export branding baseline.

## Objectives

- Keep PDF output plain, readable, and robust in constrained runtime environments.
- Include a professional brand footer with Aptale/TradeWeaver text.
- Always include the liability disclaimer in the document footer area.

## Footer Contract

- Footer brand line includes:
  - `TradeWeaver`
  - `Generated by Aptale - TradeWeaver`
- Footer disclaimer line includes the exact disclaimer from
  `landed_cost_output.disclaimer`.

## Layout Rules

- Use standard built-in PDF fonts only.
- Avoid heavy styling, external assets, or complex rendering features.
- Present key landed-cost breakdown values clearly with local currency labeling.

## Runtime Notes

- The selected PDF library is `reportlab`.
- If the library is unavailable, fail fast and surface an explicit error.
